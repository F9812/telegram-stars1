Как играть?

Основная механика

Кликайте по центральному ядру для генерации энергии. Используйте энергию для покупки генераторов, которые производят энергию автоматически.

Улучшения

Генераторы: Автоматически производят энергию каждую секунду.

Множители: Увеличивают эффективность всех генераторов.

Престиж

Когда достигнете 1,000,000 энергии, сможете выполнить престиж. Вы получите очки престижа, которые дают постоянный множитель бонуса.
